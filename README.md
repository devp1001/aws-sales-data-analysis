# AWS Serverless Sales Data Analysis Pipeline

This project automatically analyzes sales CSV data using AWS S3, Lambda, and SNS.

## Features
- Upload CSV → automatic processing
- Calculates KPIs
- Weekly analysis
- Sends email notification

## AWS Services
- S3
- Lambda
- SNS

## How it works
Upload CSV → Lambda triggers → Analysis → Output JSON → Email
